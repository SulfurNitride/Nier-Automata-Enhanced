# Nier-Automata-Enhanced

This is a simple modlist designed to enhance the gameplay of Nier Automata via graphics. 

If you need more help or have any questions please join the [NaK Discord](https://discord.gg/9JWQzSeUWt)

This modlist is based off this [Nexus Post](https://www.nexusmods.com/nierautomata/mods/29), it was not designed with MO2 in mind so I have made changes accordingly. As well as adding a few fixes of my own and a few mods.

# Requirements

The latest version of [NieR:Automata™ Game of the YoRHa Edition](https://store.steampowered.com/app/524220/NieRAutomata/) from steam. 

### System Requirements

As of right now, I have only been able to do testing at 4K with a 9070 XT and a 9800X3D and the game runs at a locked 60FPS. This modlist should run at 60FPS with anything over a GTX 1080, and 16GBs of RAM if you are targeting 1080p.

# Pre Installation

Before starting the install ensure you have done the following:
* A clean install of your the game on steam.
* And that [Visual C++](https://aka.ms/vs/17/release/vc_redist.x64.exe) is installed

# Starting the installation

1. **Install Wabbajack:**
   - Download and install [Wabbajack](https://www.wabbajack.org/).
2. **Follow the Wabbajack Installation Procedure:**
   - Refer to the section [_Installing a Modlist_](https://wiki.wabbajack.org/user_documentation/Installing%20a%20Modlist.html) in the Wabbajack Documentation.

# Running MO2.

1. First, navigate/go to the install location you picked when installing the modlist.
2. Find and run Mod `ModOrganizer.exe`.
3. Upon launching the modlist you will need to press the build button in the root builder plugin. See the image below on how to build it.
<img width="3837" height="1950" alt="image" src="https://github.com/user-attachments/assets/46a266fa-5357-4ecb-902b-59638731bdf7" />

4. Pick a reshade or turn it off and pick an android body. Vist the [Nexus Mod Page](https://www.nexusmods.com/nierautomata/mods/30) and pick the one you prefer.
5. After you have selected everything you want. You should be good to press play in MO2.

# Common Issues
As of right now I know of only one issue. Upon launching the game for the first time when you install the modlist the game will not open. To fix this go to task manager and end task for Nier Automata, afterwards if you press play again the game should open.
